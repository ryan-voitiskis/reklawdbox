## Summary

The reference section documents the MCP tools and integration contracts needed for automation.

## Steps

1. Confirm required runtime command: `./target/release/reklawdbox`.
2. Configure host environment variables from `mcp-config.example.json`.
3. Use tool-level docs to script repeatable operations.
4. Confirm response contracts before integrating downstream consumers.

## Examples

Frequently used tools:

- `read_library`
- `search_tracks`
- `update_tracks`
- `preview_changes`
- `write_xml`
