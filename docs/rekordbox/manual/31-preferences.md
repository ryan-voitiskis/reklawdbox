# [Preferences] window

To change the settings, click (gear icon) on the upper-right of the screen, or select [Preferences] from the [File] menu on the upper screen to display the [Preferences] window.
Depending on your rekordbox version and subscription plan, some items may not be displayed.

### [Configuration]

<!-- dprint-ignore -->
| | |
|---|---|
| [Audio] | Set the audio device to output sound. |
| [Sample Rate] | Set the sampling frequency of audio data sent to an audio device. A higher sampling rate brings the playback quality closer to the original sound. | [Buffer size] | Set the number of samples each time to be sent to an audio device. A larger buffer size reduces audio skipping. However, the transmission delay (latency) increases, and the scratching is delayed. |
| [Metronome] | Set the tone and volume of the metronome while you are monitoring beat grids of music files. |
