# Searching for a track

You can search for a track by following procedures.

- "Using Collection Filter" (page 27)
- "Using the category filter" (page 29)
- "Sorting track information" (page 29)
- "Using My Tag" (page 30)
- "Using RELATED TRACKS" (page 32)
- "Using Track Suggestion" (page 35)

Using playlists, you can organize tracks and select a track easily. For more information, see "Using a Playlist" (page 39).

## Using Collection Filter

1. Click [Collection] in [Media Browser].
   Collection filter search category menu ([Date Added], [Genre], [Artist], and [Album]) is displayed in the tree view.

   [Screenshot: Collection tree view showing filter categories - All, All Audio, Date Added, Genre, Artist, Album, All Videos. The Genre category is selected, showing subcategories: All, Drum'n Bass, Electro, House, Jazz, Pop, R&B]

2. Select a category.
   Tracks, which are under the conditions selected in respective search category menu, are displayed in the track list.

**Note**

- Depending on your subscription plan and DJ equipment connected to your computer, some functions may not be available. For details on subscription plan, refer to the rekordbox website.

## Using the search filter

1. Click [Collection] in [Media Browser].
   Tracks in [Collection] are displayed in the track list.

2. Click the search icon in the search filter.
   Searchable columns are displayed.

   [Screenshot: Collection view with search filter expanded showing searchable column checkboxes - All (checked), Track Title, Artist, Album, Year, BPM, Composer, Album Artist, Remixer, Label, Comments, Original Artist, Mix Name]

3. Select a column.

4. Enter characters in the search filter from your computer keyboard.
   To return to the previous view, delete the characters you have entered or click the clear button.

## Using the category filter

1. Click [Collection] in [Media Browser].
   Tracks in [Collection] are displayed in the track list.

2. Click the category filter button on the upper side of the track list.
   Category columns ([Genre], [Artist], [Album]) are displayed.

   [Screenshot: Collection view with category filter showing three columns - Genre (All/12 genres, Drum'n Bass, Electro, House), Artist (All/23 artists, Chris Mansfield, Dave Griffin, Dave Schultz), Album (All/23 albums, 1, All Through The Night, As Good As It Gets)]

3. Select a category.
   Tracks, which are under the conditions selected in respective category columns, are displayed.

## Sorting track information

1. Click [Collection] in [Media Browser].
   Tracks in [Collection] are displayed in the track list.

2. Click a column to be sorted.
   The track order is changed.
   Each time you click the column, the ascending / descending orders are switched.

3. Enter characters from your computer keyboard (Alphabet Search).
   Tracks, whose names start with the characters entered, are highlighted and selected.
   You can enter multiple characters.

**Note**

- In the PERFORMANCE mode, Alphabet Search cannot be used. Instead, use the keyboard shortcut function. The keyboard shortcut function can be set on the [Preferences] window (page 238).

## Using My Tag

You can refine your search by creating tags on tracks.
Browsing can be customized by creating tags to define the tracks' features that will enhance on your DJ performance.

- The My Tag attached to the track is displayed in the My Tag column.
- You can refine your search using a My Tag filter by setting My Tag as the search criteria.

Click the My Tag button on the right side of the screen to display the My Tag configuration window.

[Screenshot: My Tag configuration window showing a category named "Genre" with My Tag checkboxes for: Acid House, Deep House, Techno, Nu Disco, Electro House, Bass Music, Trap. Below a "Components" section with checkboxes for: Synth, Vocal, Beat, Sub Bass, Percussion, Piano. Numbered callouts: 1 = Category name, 2 = My Tag checkbox, 3 = My Tag name, 4 = Adding My Tag button]

1. Category name: You can change to a category name of your choice.

2. My Tag checkbox: Attach or remove the track's tag.

3. My Tag: A tag to attach to the track and can be changed to a name of your choice.

4. Adding My Tag: Click this to create a new tag to My Tag.

### To change a category name

Click the category name, enter new category name from your computer keyboard, and then press the [Enter] key.

### To change a My Tag name

Select My Tag, enter new tag name from your computer keyboard, and then press the [Enter] key.

### To attach My Tag to a track

Select a track, and then click My Tag checkboxes. You can attach multiple My Tags at once.

To show My Tag in [Comments] of the track list, click the settings button, open the [Preferences] window > [Advanced] category > [Browse] tab > [My Tag], and select [Add "My Tag" to the "Comments"].

### To delete a My Tag

Right-click a tag in a My Tag, and then select [Delete].

### To search for a track using My Tag (Track Filter)

1. Click the Track Filter button on the left side of the search filter.
   The track filter is displayed. In addition to My Tag, you can refine your search by [BPM], [KEY], [RATING], and [COLOR].

   [Screenshot: Track filter panel showing filter categories - BPM (All, 2, 79, 81), KEY (All, Abm, 8, Ebm), RATING (star ratings), COLOR (Pink, Red, Orange, Yellow), Genre (Acid House, Deep House, Techno, Beat), Components (checkboxes), Situation (checkboxes), Untitled (checkboxes). Each category has AND/OR toggle options. Track list shown below with filtered results.]

2. Click the required category checkboxes, and then select values, a tag, etc.
   When selecting [BPM] or [KEY], click and turn on [MASTER PLAYER] or [MASTER DECK]. The value is set as the [BPM] or [KEY] of the track loaded on the Master Player at that time.

**Hint**

- Multiple My Tags can be selected. Select [AND] and [OR] for your desired search conditions.
- When multiple My Tags are selected, tracks containing all selected My Tags are displayed in the track list.
- To cancel conditions set by My Tag filters, click [RST].

## Using RELATED TRACKS

You can display a list of tracks related to the loaded track. This helps you to select a track easily. The relation can be set with [BPM], [Key], [Matching], [Tracks in the same genre], [Ratings], My Tag, etc.

[Screenshot: RELATED TRACKS view showing tree view with conditions (BPM + KEY, Same Artist), search target (Collection), Rank column with numbered tracks, track list with columns for Track Title, Artist, Album, Genre, BPM. Numbered callouts: 1 = RELATED TRACKS conditions, 2 = Search target, 3 = Rank, 4 = Track selection, 5 = Customizing the criteria]

1. RELATED TRACKS conditions
   - You can create, edit, or delete a list of RELATED TRACKS.

2. Search target
   Select a folder or playlist for searching RELATED TRACKS.

3. [Rank]
   Tracks are ranked in the relevant order based on the conditions.

4. Track selection
   Select a deck loaded with the track, which is based for RELATED TRACKS to be displayed.

5. Customizing the criteria
   The criteria items are displayed, and then customize them.

**Hint**

- In the [2 PLAYER] and [DUAL PLAYER] layout of the EXPORT mode, or 2 deck mode of the PERFORMANCE mode, click the matching button to register 2 tracks loaded as related tracks ([Matching]).
- In the PERFORMANCE mode, click the related tracks button on the right side of the horizontal waveform or the expand button on the bottom side of the vertical waveform to display the RELATED TRACKS panel.

### Using a folder

When you create multiple RELATED TRACKS condition, you can use folders to organize conditions.

1. Right-click [Related Tracks] on the tree view, and then select [Create New Folder].
   [Untitled Folder] is added under [Related Tracks].

2. Enter the folder name using the computer keyboard, and then press the [Enter] key.

3. Drag & drop the RELATED TRACKS condition into the created folder.
   The RELATED TRACKS condition is moved into the created folder.

**Hint**

- To delete a folder, right-click the folder, and then select [Delete Folder]. You can also delete a folder by pressing the [Delete] key on the computer keyboard. All RELATED TRACKS conditions and folders included on the folder are deleted.

### Creating a new RELATED TRACKS condition

1. Click [+] on the right side of [Related Tracks] or the RELATED TRACKS folder in the tree view.
   A new RELATED TRACKS condition is added under [Related Tracks], and then the criteria setting screen is displayed.

2. Customize the criteria.

3. Click [OK].

**Hint**

- To delete a RELATED TRACKS condition, right-click the condition, and then select [Delete "Related Tracks"]. You can also delete a condition by pressing the [Delete] key on the computer keyboard.

### Using the Subpanel

Click the subpanel button to display the RELATED TRACKS Subpanel. You can display tracks of the collection or playlist on the track list with displaying Related Tracks on the Subpanel.

[Screenshot: RELATED TRACKS Subpanel showing BPM + KEY condition, with LIST and MASTER buttons, deck selectors 1-4, track name "Driving Me Crazy". Collection selected as search target with Auto checkbox. Track list showing tracks with BPM and Key values. Numbered callouts: 1 = Search target, 2 = Current RELATED TRACKS condition, 3 = Track selection, 4 = Customizing the criteria]

1. Search target
   Select a folder or playlist for searching RELATED TRACKS.

2. Current RELATED TRACKS condition
   To switch the RELATED TRACKS condition, click to display a list of conditions, and then select the condition.

3. Track selection
   [LIST]: Display RELATED TRACKS based on the track selected on the track list.
   [MASTER], [1] to [4], [A] to [B]: Select a deck loaded with the track, which is based for RELATED TRACKS to be displayed.

4. Customizing the criteria
   The criteria items of the current RELATED TRACKS are displayed, and then customize them.

**Hint**

- You can also display the RELATED TRACKS Subpanel by right-clicking the RELATED TRACKS condition and selecting [Display on Subpanel].

## Using Track Suggestion

You can display a list of recommended tracks among the tracks loaded on the deck, which can help you when selecting tracks.

[Screenshot: Track Suggestion view in EXPORT mode showing tree view with Track Suggestion conditions (Collection Radar selected, Streaming Radar, Era, Mood, Association), search target (Collection), Rank column with numbered tracks, track list with Track Title, Artist, Album, Genre, BPM columns. Numbered callouts: 1 = Recommended conditions, 2 = Search target, 3 = Rank, 4 = Tracks to select, 5 = Customizing conditions]

1. Recommended conditions of Track Suggestion
   - Collection Radar: Displays tracks with similar musical characteristics to the currently playing track from [Collection] from the top.
   - Streaming Radar: Displays tracks with similar musical characteristics to the currently playing track from a streaming service from the top.
   - Era: Displays tracks released in a similar era to the currently playing track from the top.
   - Mood: Displays tracks with the similar vibe to the currently playing track from the top.
   - Association: Displays tracks with the same composer, label, etc. as the currently playing track from the top.

   **Note**

   - To use [Collection Radar] or [Streaming Radar], open the [Preferences] window > [Analysis] category > [Track Analysis] tab, and enable [Rader Analysis].
   - For tracks which have already been imported, another analysis data is required to use [Collection Radar] or [Streaming Radar]. Right-click the track and select [Add New Analysis Data]. (When analysis data is added, the analysis icon will appear in the status column.)

2. Search target
   Select the folder, playlist, etc. in which you want to search for the recommended tracks.

   **Note**

   - When using [Streaming Radar], the search targets are the tracks from streaming services, so you cannot select the search target.

3. [Rank]
   Ranks in order of recommendation.

4. Tracks to select
   Select the deck loaded with the track for the recommended track.

5. Customizing conditions (Only for [Collection Radar] and [Streaming Radar])
   Click to display the setting window for [Collection Radar] and [Streaming Radar], where you can change the conditions.

   [Screenshot: "Customize conditions of tracks to recommend" dialog window showing options for Streaming service (SoundCloud, Beatport, Beatsource, TIDAL tabs), BPM checkbox (+-5), Key dropdown (Related key 1), Genre section with "Same genre" checkbox and genre grid (House, Techno, Trance, Hardcore, Acodance, UK Garage, EDM, Game, Drum & Bass, Dubstep, Retro, Future Bass, Breaks, R&B/Soul, Hip-Hop, Orchestral, Chill, Jazz, Pop, Reggae, Piano, Classical, Blues, Rock, Metal), Vocal dropdown (Song including vocals), with OK and Cancel buttons]

   - Streaming service: Set the information source for recommended tracks.
   - BPM: Set up BPM.
   - Key: Set up Key.
   - Vocal: Set vocals included/not included.

### Using the Subpanel of Track Suggestion

Click the Track Suggestion subpanel button to display the Track Suggestion Subpanel. You can display recommended tracks under the conditions selected on the Subpanel with displaying tracks of the collection or playlist on the track list.

[Screenshot: Track Suggestion Subpanel showing Collection Radar condition, with LIST and MASTER buttons, deck selectors 1-4 and A-B, track name "Driving M..." Collection selected as search target with Auto checkbox. Track list showing recommended tracks with BPM and Key values. Numbered callouts: 1 = Recommended conditions, 2 = Search target, 3 = Tracks to select, 4 = Customizing conditions]

1. Recommended conditions of Track Suggestion
   - Collection Radar: Displays tracks with similar musical characteristics to the currently playing track from [Collection] from the top.
   - Streaming Radar: Displays tracks with similar musical characteristics to the currently playing track from a streaming service from the top.
   - Era: Displays tracks released in a similar era to the currently playing track from the top.
   - Mood: Displays tracks with the similar vibe to the currently playing track from the top.
   - Association: Displays tracks with the same composer, label, etc. as the currently playing track from the top.

   **Note**

   - To use [Collection Radar] or [Streaming Radar], open the [Preferences] window > [Analysis] category > [Track Analysis] tab, and enable [Rader Analysis].
   - For tracks which have already been imported, another analysis data is required to use [Collection Radar] or [Streaming Radar]. Right-click the track and select [Add New Analysis Data]. (When analysis data is added, the analysis icon will appear in the status column.)

2. Search target
   Select the folder, playlist, etc. in which you want to search for the recommended tracks.

   **Note**

   - When using [Streaming Radar], the search targets are the tracks from streaming services, so you cannot select the search target.

3. Tracks to select
   [LIST]: Display recommended tracks for the selected track in the track list.
   [MASTER], [1]-[4], [A]-[B]: Select the deck loaded with the track for the recommended track.

4. Customizing conditions (Only for [Collection Radar] and [Streaming Radar])
   Click to display the setting window for [Collection Radar] and [Streaming Radar], where you can change the conditions.

   [Screenshot: "Customize conditions of tracks to recommend" dialog window showing same options as previous customize dialog - Streaming service tabs, BPM, Key, Genre grid, and Vocal settings with OK and Cancel buttons]

   - Streaming service: Set the information source for recommended tracks.
   - BPM: Set up BPM.
   - Key: Set up Key.
   - Vocal: Set vocals included/not included.

**Hint**

- You can also display the Subpanel by selecting Track Suggestion in the tree view, right-clicking it, and clicking [Show in subpanel].
